This directory contains a few non-essential files from the JB system that the JANET tool uses at the run time. JB is a public domain parser and lexer generator for Java. JB has been developed by Dennis Heimbigner at the University of Colorado.

JANET has been developed with aid of the JB. To build JANET from source files, you need full JB package. It can be downloaded from sites given above. Files in this directory are those used by the JANET tool at the run time only.
